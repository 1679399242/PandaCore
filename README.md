## Requirements

+ Platform: Linux (Ubuntu 16.04), Windows
+ Processor supporting SSE2 
+ ACE = 6.3.3 (5.8.3 included for Windows) 
+ MySQL = 5.6 32Bit
+ CMake = 3.12.4 
+ OpenSSL = 1.0.2g 32Bit
+ GCC = 5.4.0 (Linux only) 
+ MS Visual Studio = 17 (2017)(Windows only)
